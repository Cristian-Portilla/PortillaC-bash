Interprete Interactivo de Comandos

Comando y Descripción 
`exit` / `salir`  : Cierra el bash 
`cd <directorio>` : Cambia el directorio actual 
`pwd` : Muestra la ruta actual
`ls` : Lista archivos (implementación con `execl`) 
`mkdir <nombre>` : Crea un directorio 
`rm <archivo>` : Elimina un archivo 
`cp <origen> <destino>` : Copia archivos (implementación manual) 
`mv <origen> <destino>` : Mueve o renombra archivos 
`cat <archivo>` : Muestra contenido de archivos
